# STM32平衡车项目
出于兴趣自制的基于STM32F103RE的平衡车。  
使用了FreeRTOS+LVGL。   
GUI的PID参数设置的是循迹转向环，所以全是0.0，需要的请自行更改。  
没写断电保存

软件上使用了HAL库。 
**工具使用了：**
+ STM32CUBEMX 初始化硬件
+ Vscode
+ Vscode的Keil Assistant插件
+ SquareLine Studio 设计GUI。
